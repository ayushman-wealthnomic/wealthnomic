SwarmOS

The Open Source Framework for Multi-Agent Financial Swarms. Built on LangGraph and FinRobot to orchestrate autonomous research, due diligence, and execution.

The Thesis

Standalone agents are limited. Swarms are exponential. In institutional finance, decisions are rarely made by one person. Analysis requires a consensus between fundamental analysts, technical researchers, and risk managers.

SwarmOS brings this collaborative intelligence to AI. By leveraging LangGraph’s stateful orchestration and FinRobot’s specialized financial tools, we create "digital investment committees" that debate data before reaching a trading decision.

Multi-Agent Orchestration

LangGraph FinRobot ReAct Paradigm Stateful Memory Tool Use Consensus Voting

Sample Swarm Definition (Python)

# Define a stateful financial swarm from swarmos import SwarmGraph, FinancialAgent market_researcher = FinancialAgent(role="analyst", tools=["yfinance", "fmp"]) risk_manager = FinancialAgent(role="risk", tools=["portfolio_var"]) workflow = SwarmGraph() workflow.add_node("research", market_researcher) workflow.add_node("risk_check", risk_manager) workflow.add_edge("research", "risk_check") workflow.compile().invoke({"ticker": "NVDA"})
