RL Gymnasium

The open-source Reinforcement Learning environment for financial markets. Training agents to trade, provide liquidity, and manage risk in high-fidelity simulations.

The Thesis

Backtesting is not training. Static historical data only tells you what *would* have happened if you were a passive observer. In real markets, your actions have impactâ€”slippage, price pressure, and order book dynamics all matter.

RL Gymnasium provides a dynamic playground for AI agents. By integrating FinRL with OpenAI Gymnasium and PettingZoo, we enable multi-agent reinforcement learning where bots compete and cooperate in simulated exchange environments.

Framework Integration

FinRL Gymnasium PettingZoo PyTorch Ray RLib SB3

Our gymnasium provides pre-built environments for:

  • Portfolio Allocation: Optimizing weights across a basket of assets.
  • Market Making: Managing bid-ask spreads and inventory risk.
  • Execution Tracking: Slicing large orders to minimize market impact.
  • Adversarial Markets: Training agents to survive against "shark" algorithms.

Technical Specifications

Standards OpenAI Gym / PettingZoo (MARL)
Optimizers PPO, DDPG, SAC, TD3
Simulation Event-driven with Order Book matching
Latency Sub-millisecond step updates
Hardware Distributed GPU training via Ray
License Apache 2.0

Sample Training Script (Python)

# Initialize a FinRL environment for Nifty 50 from rl_gym import StockTradingEnv from stable_baselines3 import PPO env = StockTradingEnv(df=market_data, initial_amount=100000) model = PPO("MlpPolicy", env, verbose=1) model.learn(total_timesteps=1000000) model.save("ppo_trading_agent_v1")
